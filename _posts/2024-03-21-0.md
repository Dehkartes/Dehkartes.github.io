# 이력서 준비 2: 자기소개서
1. 자기소개서의 역할
	- 지원자를 요약
	- 핵심 경험과 역량을 요악해서 서술
2. 자기소개 주요 요소
	- 개발 관련 경험
		- 기능 구현 경험
		- 기술 적용 경험
	- 간결함
		- 5 ~ 7 줄 내외
	- 실제 사례 어필
		- 수상 내역
		- 성과
		- 구체적 수치 제시
	- 강점 키워드
		- 의사소통 능력
		- 문제 해결력
3. 주의사항
	- 설득력 있는 결과를 제시
		- 1일 1커밋인데 무의미한 내용만 있음 → 끈기에 대한 설득력 없음
	- 당연한 것은 강점이 아님
		- 일을 끝까지 해냄
		- 지각을 하지 않음
	- 호불호 갈리는 특성은 제외할 것
		- 해결되지 않는 문제를 풀릴때까지 파고듬
		- 먼저 도전해보고 동료들을 설득
		- 모든 의견을 들어보고 다수가 만족하는 방향을 선택
