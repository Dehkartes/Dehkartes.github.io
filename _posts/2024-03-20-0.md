# 이력서 준비
	1. 경험정리
		- 형태 상관없이 경험 정리
		- 분류
			- 기능구현
			- 트러블 슈팅
			- 기능/개선 결과별
		- 골든 서클
			- Why(목적), How(과정), What(결과) 으로 경험 서술

	2. 구문 최적화
		- 문장 간결화
			- 불필요한 동사, 접속사, 조사등을 제외하고 어색하지 않은지 확인
			- 하나의 문장에는 하나의 내용만 포함
			- 문단에 대한 두괄식 제목 선정
			- 개조식으로 세부사항 서술
		- 불필요한 정보 줄이기
			- 기술 약어(AI, CI/CD) 사용
			- 경험 전달에 필용하지 않은 기술적 특징 제거
		- 1인칭, 2인칭 표현 제거
			- 제가, 저는, 우리가~
		- 수동태, 변역체, 구어체 사용 자제
			- ex) 개선 되었다(피동) -> 개선 했다(능동).
			- ex) 하려고(구어체) -> 하기 위하여(문어체)
