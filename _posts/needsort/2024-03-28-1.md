# 문자열
- string (str)
- str(): 형변환
- strA in strB: strA가 strB의 연속된 부분 문자열이면 True, 아니면 False 반환
- string.replace(old, new): 문자열의 old를 new로 변환

# 정수
- integer (int)
- int(): 형변환

# 리스트
- 배열
- append(x): x 객체를 배열 마지막에 추가

# 셋
- 집합, 중복을 허용하지 않음

# 딕셔너리
